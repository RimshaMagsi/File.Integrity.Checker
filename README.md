# 🔒 File Integrity Checker

A cybersecurity tool built with Python and Streamlit that generates and verifies SHA-256 hashes to ensure file integrity and detect unauthorized modifications.

## 📌 Features

- Generate SHA-256 hash for any file
- Verify file integrity
- Detect file tampering or modifications
- User-friendly Streamlit interface
- Download generated hash

## 🛠 Technologies Used

- Python
- Streamlit
- Hashlib (SHA-256)

## 🚀 How It Works

1. Upload a file.
2. The application generates a SHA-256 hash.
3. Copy or download the generated hash.
4. Upload the file again and enter the original hash.
5. The application verifies whether the file has been modified.

## 📷 Example Use Cases

- File integrity verification
- Digital forensics
- Cybersecurity learning
- Security awareness demonstrations
